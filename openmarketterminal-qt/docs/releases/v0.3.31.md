# v0.3.31 - Honest Data Menus and Day-One Crash Fixes

- Fixes crashes present since the first release: opening Markets > GOVT Data
  aborted the app (invalid Qt connection flags in the Hong Kong, Australia,
  and UK panels), and the Agent Config screen's System tab carried the same
  fault. All connection sites audited.
- Economics now shows real data with zero API keys: the FRED panel's
  flagship series (GDP, real GDP, CPI, core CPI, unemployment, fed funds,
  2y/10y Treasury yields) transparently fall back to their original
  sources' DBnomics mirrors (BEA, BLS, Federal Reserve) when no FRED key is
  configured. Add a free FRED key in Settings for the full catalogue.
- Removes the AKShare screen (China-market data browser inherited from the
  upstream project) and prunes Gov Data from ten providers to the six with
  first-class pipelines: US Treasury, US Congress, UK, France, Hong Kong,
  and Australia — the UK panel is reachable for the first time.
- Everything in v0.3.30 is included.

This release keeps user credentials, account data, journals, and local model
configuration on the user's machine. No API keys or secrets are included in
the release artifacts or source tree.
