# v0.3.22 — READ FULL: real reader-mode extraction

READ FULL was failing on plenty of non-paywalled sites (e.g. Wired) because the
headless browser chokes on heavy pages. It now works like a reader/"reading
list" tool: it fetches the page over plain HTTP and extracts the readable
article text (JSON-LD article body, with a readability fallback) — fast and
reliable. The headless browser is now only a backup for sites that need JS
rendering. The extracted text still feeds ANALYZE.
