# v0.3.29 - Crypto Window Overhaul and Spot-Style Engine

- Rebuilds the crypto trading screen around the connected exchange: native
  quote pairs (Coinbase shows real /USD markets), live account holdings in the
  positions tab on spot venues, and 24h market stats replacing perp-only
  funding cards.
- Adds an authenticated account stream: instant blotter updates on fills and
  cancels, deduplicated order notifications, and the trader's own resting
  orders and estimated average entry drawn on the DOM ladder.
- Makes paper trading honest: market orders walk the live order book (partial
  fills on thin depth) and refuse to fill against a stale or missing book.
- Adds local price alerts (right-click a watchlist row) that persist across
  restarts and never re-fire stale triggers.
- Gives the scalp engine a real --style scalp|spot switch with spot presets
  for larger intrahour moves, and a cockpit that truthfully reports the
  configured style, feed health, and cost hurdle.
- Migrates all Coinbase market data to the Advanced Trade API (REST and
  WebSocket) ahead of the legacy Exchange API sunset, with a fallback path.
- Computes exact realized P&L for netted (mixed yes/no) Kalshi settlements and
  hardens account-reconciliation freshness handling in the advisor safety
  gate.

This release keeps user credentials, account data, journals, and local model
configuration on the user's machine. No API keys or secrets are included in
the release artifacts or source tree.
