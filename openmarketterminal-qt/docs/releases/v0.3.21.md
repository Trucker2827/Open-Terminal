# v0.3.21 — News: on-device transcription for caption-less videos

TRANSCRIBE now handles videos that have **no captions**: it downloads the audio
and transcribes it locally with **Apple's on-device Speech recognition** — no
model download, private, free. (Captioned videos still use the faster caption
path.) The transcript feeds ANALYZE just like captions/article text.

First use prompts once for **Speech Recognition** permission (System Settings →
Privacy & Security). macOS-only; sources that block automated audio download
(e.g. Bloomberg) still can't be fetched.
