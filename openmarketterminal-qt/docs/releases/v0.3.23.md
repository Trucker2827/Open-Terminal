# v0.3.23 — Local lock: 4-digit PIN + Touch ID

Replaces the meaningless "Log in / Log out" (there's no cloud account) with a
real local lock:
- The top-bar button is now **LOCK** — it locks the terminal (Touch ID or PIN to
  return). With no PIN yet, pressing LOCK starts **4-digit PIN** setup.
- **Touch ID** unlock on Mac: the lock screen auto-prompts for your fingerprint,
  with the PIN as fallback.
- PIN is now **4 digits** (was 6). Existing PINs are cleared once so you set a
  fresh 4-digit one.
