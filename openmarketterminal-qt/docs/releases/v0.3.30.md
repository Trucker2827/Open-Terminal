# v0.3.30 - Windows Restored, Spot-Style Engine, Exact Kalshi P&L

- Restores the Windows installer (v0.3.29 shipped macOS/Linux only): the
  oversized CLI dispatcher unit that crashed the MSVC compiler front-end is
  now split into small, independently compiled units.
- Adds a --style scalp|spot switch to the crypto automation engine with spot
  presets tuned for larger intrahour moves, persisted and reported truthfully
  in the cockpit alongside colored daemon/WebSocket status chrome.
- Shows real account holdings in the crypto positions tab and 24h market
  stats in the market tab on spot venues (Coinbase Advanced), replacing
  perpetuals-only fields that always rendered empty.
- Computes exact realized P&L for netted (mixed yes/no) Kalshi settlements
  and tolerates normal account-reconciliation cadence in the advisor safety
  gate, unblocking automated evidence collection.
- Everything in v0.3.29 is included: the crypto window overhaul, the
  authenticated account stream, honest paper fills, local price alerts, and
  the Coinbase Advanced Trade API migration.

This release keeps user credentials, account data, journals, and local model
configuration on the user's machine. No API keys or secrets are included in
the release artifacts or source tree.
